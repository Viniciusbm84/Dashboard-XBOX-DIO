# 🎮 XBOX Game Pass Dashboard - Projeto DIO

## 📋 Descrição do Projeto

Este projeto foi desenvolvido como parte do bootcamp da **Digital Innovation One (DIO)**, com o objetivo de criar um dashboard interativo e visualmente atraente para análise de dados de assinaturas do Xbox Game Pass.

---

## 🎯 Objetivo

Criar um dashboard profissional no Excel utilizando:
- Paleta de cores oficial do Xbox
- Análise de dados de assinantes
- Visualizações gráficas interativas
- KPIs relevantes para gestão de assinaturas

---

## 📊 Estrutura do Arquivo

O arquivo Excel contém as seguintes abas:

| Aba | Descrição |
| --- | --- |
| **Assets** | Paleta de cores Xbox (#9BC848, #22C55E, #2AE6B1, #5BF6A8) e zona negativa (#E8E6E9) |
| **Bases** | Base de dados com 295 assinantes contendo informações completas |
| **Dashboard** | Dashboard visual com KPIs, tabelas e gráficos |
| **README** | Documentação do projeto (esta aba) |

## 📁 Dados da Base

A base de dados (aba Bases) contém as seguintes colunas:

| Coluna | Descrição |
| --- | --- |
| Subscriber ID | ID único do assinante |
| Name | Nome do assinante |
| Plan | Plano (Ultimate, Standard, Core) |
| Start Date | Data de início da assinatura |
| Auto Renewal | Renovação automática (Yes/No) |
| Subscription Price | Preço da assinatura |
| Subscription Type | Tipo (Monthly, Quarterly, Annual) |
| EA Play Season Pass | Possui EA Play (Yes/No) |
| EA Play Season Pass Price | Preço do EA Play |
| Minecraft Season Pass | Possui Minecraft (Yes/No) |
| Minecraft Season Pass Price | Preço do Minecraft |
| Coupon Value | Valor do cupom de desconto |
| Total Value | Valor total da assinatura |

## 🛠️ Tecnologias Utilizadas

- **Microsoft Excel** - Criação do dashboard
- **Office.js** - Automação e formatação
- **Fórmulas Excel** - CONT.SE, SOMASE, SUMIFS, etc.
- **Gráficos Excel** - Pizza, Rosca, Barras Clusterizadas

---

## 🚀 Como Utilizar

1. Abra o arquivo Excel
2. Navegue até a aba **Dashboard** para visualizar as métricas
3. A aba **Bases** contém os dados brutos para consulta
4. A aba **Assets** contém a paleta de cores de referência
5. Modifique os dados na aba Bases e veja o dashboard atualizar automaticamente
